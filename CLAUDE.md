# CLAUDE.md

This project uses the **Karpathy Behavioral Guidelines** for all coding agents.

**Please read and follow the demarcated "Karpathy Behavioral Guidelines" section inside [AGENTS.md](./AGENTS.md).**

These guidelines emphasize:
- Thinking before coding and surfacing assumptions early
- Extreme simplicity (implement only what was asked)
- Surgical, minimal edits with no unrelated changes
- Goal-driven execution with verifiable success criteria

The pure guidelines are also available standalone in [`karpathy-guidelines.md`](./karpathy-guidelines.md) for easy appending to existing agent instruction files.

This content (or a reference to it) should be applied consistently whether you are Claude Code, Cursor, Copilot, Grok Build, or another agent.
